# Руководство пользователя EduSphere

**Стандарт:** ГОСТ 19.505-79  
**Форма:** Интерактивная справка (встроена в платформу) + PDF

---

## 1. Введение

EduSphere — образовательная платформа для учеников, преподавателей и администраторов. Система поддерживает адаптивное обучение, индивидуальные учебные планы и AR-классы.

### Роли

| Роль | Возможности |
|------|------------|
| **Ученик** | Просмотр материалов, генерация плана, AR-классы, просмотр оценок |
| **Преподаватель** | Управление материалами, утверждение планов, выставление оценок |
| **Администратор** | Аналитика, управление системой |

---

## 2. Начало работы

### 2.1. Регистрация

1. Откройте `http://localhost:3000`
2. Нажмите «Зарегистрироваться»
3. Заполните: имя, email, пароль (≥8 символов), роль
4. Нажмите «Регистрация»

### 2.2. Вход

1. Введите email и пароль
2. Нажмите «Войти»

---

## 3. Создание учебного плана (Конструктор)

### Для ученика

1. Перейдите в раздел **«Мой план»**
2. В поле «Цели обучения» опишите, чему хотите научиться
3. В поле «Интересы» укажите темы через запятую
4. Нажмите **«Сгенерировать план»**
5. Система создаст индивидуальную программу с помощью ИИ
6. Дождитесь утверждения преподавателем

### Для преподавателя

1. Перейдите в **«Учебные планы»**
2. Просмотрите планы со статусом `AI_GENERATED`
3. При необходимости внесите правки
4. Нажмите **«Утвердить»** → статус `TEACHER_APPROVED`

---

## 4. Проведение AR-сессии

### Веб-версия

1. Перейдите в **«AR-класс»**
2. Выберите активную сессию
3. Нажмите **«Присоединиться»**

### Мобильное приложение

1. Установите EduSphere AR (Android 14+ / iOS 18+)
2. Войдите с теми же учётными данными
3. Выберите AR-класс из списка
4. Наведите камеру на горизонтальную поверхность
5. 3D-модель появится в дополненной реальности

**Требования к устройству:**
- Поддержка ARCore (Android) или ARKit (iOS)
- Камера с автофокусом
- Стабильное интернет-соединение

---

## 5. Анализ дашбордов

### Панель прогресса (ученик)

- **Средний балл** — среднее по всем предметам
- **Вовлечённость** — активность на платформе
- **Модули** — прогресс по учебному плану
- **Рекомендации ИИ** — адаптивные советы по контенту

### Панель преподавателя

- График успеваемости по предметам
- Активность учеников по неделям
- Раздел выставления оценок

---

## 6. Работа с материалами

1. Перейдите в **«Материалы»**
2. Просматривайте контент по типам: текст, видео, аудио, изображения, AR-модели
3. Материалы отфильтрованы по предмету и сложности

---

## 7. Просмотр оценок

Ученик видит оценки на панели прогресса (график по предметам).

Преподаватель выставляет оценки:
1. **«Оценки»** → ID ученика, предмет, балл, комментарий
2. Нажмите **«Выставить оценку»**

---

## 8. Справка

Нажмите кнопку **«?»** в верхней панели для интерактивной справки.
